INLINE ELEMENTS

  • Inline elements flow within the line of text and only take up the space required by their content.
    Examples: span, a, img, strong, em, input, button.

BLOCK LINE ELEMENTS

  • Block elements always start on a new line and occupy the full width of their parent container.
    Examples: div, p, h1-h6, ul, ol, li, table, footer, header, section, article.

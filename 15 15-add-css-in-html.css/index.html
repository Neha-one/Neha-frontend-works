ADDING OF CSS IN HTML

  1. inline CSS 
    2.             
<p style="color: blue; background-color: aqua;">
    this is used in an HTML page using style attribute.
    </p>
  2. internal CSS

    3. it means to add a style tag & place the HTML page's style within it. after title tag.

                
  <style>
        p{
            background-color: rgb(164, 251, 251);
            color: brown;
        }
    </style>
  3. external CSS

    4. it means to create a .css file & place all your styles inside that .css file.
